Repositório pessoal de exercícios para praticar programação com rust.

Inicialmente organizei os exercícios em um projeto apenas. Minha intenção era de ter tudo em apenas um repositório.
Então achei interessante usar módulos.
Cada exercício então é feito como um módulo, que posteriormente é importado na função main, onde exibe todas as saídas dos exercícios na sequência que são inseridos.
Futuramente espero inserir um menu para decidir quais exercícios queira visualizar na execução do binário.
